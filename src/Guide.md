# updating package.json

```json
{
  "name": "customers",
  "version": "1.0.0",
  "description": "",
  "main": "src/app.js",
  "scripts": {
    "start": "node src/app.js"
  },
  "author": "",
  "license": "ISC"
}
```

# Run

```powershell
npm run start
```
